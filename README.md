# commit

Standalone commit command powered by pi skills.

## Local Setup

```bash
bun install
```

`postinstall` runs `lefthook install` automatically.

## Local Development

```bash
bun run lint
bun run lint:fix
bun run fmt:check
bun run fmt:fix
bun run typecheck
bun run test
bun run check:fast
bun run check
```
